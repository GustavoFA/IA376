# IA376 — Deep Generative Modeling

Professor: Profa. Dra. Paula Dornhofer Paro Costa  
Semester: 1st Semester 2026  
Institution: FEEC — UNICAMP

This repository contains the activities developed for the IA376 course on Deep Generative Modeling at UNICAMP.

## Activities

### A1 — Building a Model for the Synthesis of Tabular Healthcare Data

The objective of this activity is to design and implement a method to generate a synthetic database of blood count test results using real anonymized laboratory data as reference.  
The generated dataset must contain at least **1,000 fictitious exam records**, and the methodology used for the data synthesis must be clearly described and justified.
